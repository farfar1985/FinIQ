Never say FinIQ will "replace" Mars's current analytics tools or describe their setup as "fragmented". Use "augment", "consolidate", "evolve", or "enhance" instead. Describe their current state as "dispersed" or "separate" if needed.

**Why:** Mr. Savino flagged this — Mars should feel their existing investment is being built upon, not thrown away. This doc could be shared directly with the client.

**How to apply:** Any time generating or editing client-facing documents for Mars, use partnership-oriented language. This applies to SRS, proposals, presentations, and any deliverable that Mars stakeholders may read.
