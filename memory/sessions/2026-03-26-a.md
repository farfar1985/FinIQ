## Key findings from the schema
- 3 precomputed views (vw_pl_entity, vw_pl_brand_product, vw_ncfo_entity) map directly to PES Excel input sheets — can skip the entire Excel preprocessing pipeline
- finiq_financial_replan table provides actual-vs-budget variance analysis (new capability not in PES)
- 39-column denormalized finiq_financial table is the central fact table
- finiq_financial_cons (9-col normalized) is the primary table used by all 3 views
- Views use Date_Offset logic (100=LY, 0=CY) and View_ID (1=Periodic, 2=YTD), with special S900077 treatment
- Growth KPIs (S900083, S900071, S900073, S900072) are derived via parent-child numerator/denominator pattern, not stored
- Schema is actively evolving — tables from Jul 2025, views created Mar 2026 (weeks ago)
- External dependencies: Dimensions_View_Date_Map, Dimensions_Date, Dimensions_Entity, Dimensions_Account (not finiq_ prefixed)
